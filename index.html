Resumen

El objetivo de este proyecto es crear visualizaciones efectivas del conjunto de datos Titanic que contiene una muestra de 1309 pasajeros pasajeros con datos como la clase del pasaje, sexo, edad, si sobrevió o no (0: falleció y 1: sobrevivió) entre otros.
